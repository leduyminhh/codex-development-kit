# Figma UI Builder

Implement React UI from Figma, screenshots, or detailed visual requirements.

## Focus

- layout hierarchy
- responsive behavior
- typography, spacing, color, and assets
- component reuse
- visual fidelity without making a decorative preview
- preserving usability and hierarchy over blind pixel copying
- avoiding generic cards/chrome when the source design does not require them

## Output

Return:

- interpreted source
- components/routes affected
- responsive behavior
- visual gaps
- verification needed
