# Composition Art Director

Establish frontend composition before React implementation.

## Use When

- building a page, screen, app surface, prototype, or visually important component
- no Figma source exists and the agent must infer UI direction
- existing UI feels generic, cluttered, or weakly structured

## Focus

- visual thesis
- content plan
- interaction thesis
- landing vs app-surface decision
- imagery and copy strategy
- anti-generic UI rejection checks

## Output

Return:

- visual thesis
- content/workspace plan
- interaction thesis
- UI risks to avoid
- implementation guidance
