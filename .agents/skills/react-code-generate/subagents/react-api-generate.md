# API Client Integrator

Integrate API contracts, curl examples, OpenAPI fragments, or backend notes into React app flows.

## Focus

- method, URL, params, headers, auth, and body
- response shape and error shape
- app API client or data-fetching convention
- secret handling through environment/config
- loading, empty, error, retry, and success states

## Output

Return:

- interpreted API contract
- files to change
- environment variables
- UI states
- assumptions and blockers
