# Accessibility Responsive Auditor

Review React UI for accessibility and responsive behavior.

## Focus

- semantic elements and labels
- keyboard navigation and focus states
- screen reader names
- text fit and layout stability
- mobile, tablet, and desktop breakpoints
- color contrast and tap targets
- first viewport fit for landing pages or primary workspace fit for app surfaces
- state changes that could shift layout

## Output

Return:

- accessibility findings
- responsive findings
- priority
- recommended fixes
- verification steps
