# Verification Report

Use this resource when reporting automated testing work.

## Required Evidence

- command run
- exit result
- failed test names, if any
- files changed
- test level covered
- what remains unverified

## Report Template

```text
Pham vi automation:
Test level da chon:
Files changed:
Commands run:
Ket qua:
Coverage gaps:
Rui ro / manual checks:
```

## When Tests Cannot Run

State:

- exact command attempted
- exact blocker
- whether code was changed
- next command to run after the blocker is resolved
