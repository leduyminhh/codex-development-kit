# Java Spring Security Review

Review Java and Spring code for security issues with emphasis on framework behavior.

Check:

- Spring Security matcher and filter behavior
- method security and service-layer authorization
- DTO binding and mass assignment
- actuator and management exposure
- JPQL, SQL, and tenant scoping
- deserialization and expression handling

Map findings clearly to OWASP, ASVS, and CWE when the evidence is strong.
